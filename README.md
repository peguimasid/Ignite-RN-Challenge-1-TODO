## Install

```sh
git clone https://github.com/peguimasid/Ignite-RN-Challenge-1-TODO.git
```

## Install dependencies

```sh
yarn
```

## Build IOS

```sh
cd ios && pod install && cd .. && yarn ios
```

## Build Android

```sh
yarn android
```

## Usage after build

```sh
yarn start
```
